# AI Recruiter UA

[Українська](README.md) | [Русский](README.ru.md) | [English](README.en.md)

**Персональный AI-рекрутер для украинского рынка труда**, построенный на Claude Code.

Автоматизирует весь цикл поиска работы: от сканирования порталов до генерации PDF-резюме и отслеживания заявок. Поддерживает три языка: **украинский**, **русский**, **английский**.

---

## Что умеет

| Функция | Детали |
|---------|--------|
| **Оценка вакансий** | Система A-F скоринга: совпадение с CV, зарплата, культура, red flags |
| **Генерация резюме** | ATS-оптимизированный PDF под конкретную вакансию |
| **Сканирование порталов** | Djinni, DOU, Work.ua, Robota.ua |
| **Трекер заявок** | Статусы от Evaluated до Offer/Rejected |
| **Cover Letter** | Генерируется вместе с CV, привязана к JD |
| **LinkedIn аутрич** | Находит контакт в компании + черновик сообщения |
| **Follow-up кадан** | Напоминания когда и как написать после отправки |
| **Анализ отказов** | Паттерны по scoring, компаниям, ролям |

## Быстрый старт

```bash
git clone https://github.com/YOUR_USERNAME/ai-recruiter-ua
cd ai-recruiter-ua
npm install
npx playwright install chromium
```

Открыть в [Claude Code](https://claude.ai/code) или [OpenCode](https://opencode.ai).  
Вставить URL вакансии — агент запустит полный пайплайн автоматически.

---

## Настройка языка

В `config/profile.yml`:

```yaml
language:
  agent: "ru"          # язык ответов агента: uk | ru | en
  cv_default: "ru"     # язык резюме по умолчанию
  jd_output: "match_jd" # резюме языком вакансии
```

Технические термины (LLM, RAG, Tool Calling, MCP, HITL, LangGraph) — всегда на английском.

---

## Система оценки

Каждая вакансия получает оценку **1–5** по 6 блокам:

- **4.5+** → Сильное совпадение, подаваться немедленно
- **4.0–4.4** → Хорошо, стоит подаваться
- **3.5–3.9** → Неплохо, но не идеал
- **< 3.5** → Не рекомендуется
